**Root cause:** API error responses don't include `quotes` array, causing undefined.forEach()

```typescript

  setQuotes([]);
  setPagination(null);

  setQuotes(Array.isArray(data.quotes) ? data.quotes : []);
  setPagination(data.pagination || null);
}
```
