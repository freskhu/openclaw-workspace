# Social Media Ops - QA Checklist

## Pre-Flight
- [ ] PILLARS.md reviewed
- [ ] VOICE_GUIDE.md reviewed

## Content Quality
- [ ] Micro-story pattern applied (problem → experiment → result → takeaway)
- [ ] Specific details included (not abstractions)
- [ ] No hype words (leverage, disrupt, synergy, innovative)
- [ ] No CV tone ("As an X with Y experience...")
- [ ] Process emphasized over outcome
- [ ] Cross-domain connection made (where relevant)

## Voice Attributes
- [ ] Understated (no ALL CAPS, no "GAME CHANGING")
- [ ] Specific (concrete examples, not generalities)
- [ ] Slightly skeptical (questions assumptions)
- [ ] Process-obsessed (shows the iteration)
- [ ] Cross-domain (connects unlikely dots)

## Credentials
- [ ] Credibility line used at most once
- [ ] Line is subtle (not resume-style)
- [ ] Credentials not mentioned again after opening

## Confidentiality
- [ ] No client names (or permission obtained)
- [ ] No exact revenue figures
- [ ] No live trading positions
- [ ] No entry/exit prices
- [ ] No PnL numbers
- [ ] No sensitive operational details
- [ ] Generalized references used where needed

## Platform Formatting
- [ ] LinkedIn: Line breaks for readability, one hashtag max
- [ ] X Short: Under 280 chars, punchy
- [ ] X Thread: Numbered (1/8, 2/8...), one idea per tweet
- [ ] Character count checked
