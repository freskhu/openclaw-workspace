# Social Media Ops - Prompt Template

```
Create social media content with the following parameters:

CONTENT_TYPE: {{content_type}}
PILLAR: {{pillar}}
TOPIC: {{topic}}
CREDIBILITY_MODE: {{credibility_mode}}
PLATFORM: {{platform}}

REFERENCES:
- Read skills/social_media_ops_v1/PILLARS.md
- Read skills/social_media_ops_v1/VOICE_GUIDE.md
- Reference skills/social_media_ops_v1/SAMPLE_POSTS.md for patterns

TASKS:
1. Identify the core insight or story
2. Apply micro-story pattern (problem → experiment → result → takeaway)
3. Include specific details (no abstractions)
4. Apply credibility line if CREDIBILITY_MODE = include/subtle
5. Check against voice attributes (understated, specific, skeptical)
6. Verify confidentiality (no positions, no PnL, no client specifics)
7. Format for PLATFORM
8. Include pillar alignment note
9. Include voice check note
10. Include confidentiality verification

CONSTRAINTS:
- One credibility line max (see VOICE_GUIDE.md)
- No hype words (leverage, disrupt, synergy)
- No CV tone
- No confidential info
- Specific details required
- Process over outcome focus

DELIVERABLES:
- Draft post(s)
- Pillar alignment note
- Voice check note
- Confidentiality verification
```
