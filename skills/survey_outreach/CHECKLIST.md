# Survey Outreach - QA Checklist

## Survey Design
- [ ] Max 10 questions total
- [ ] No leading questions (neutral phrasing)
- [ ] No double-barreled questions (one concept per Q)
- [ ] Clear, jargon-free language
- [ ] Response options are exhaustive and mutually exclusive
- [ ] Only essential fields are required
- [ ] Progress indicator visible
- [ ] Mobile-responsive design

## Legal/Compliance
- [ ] Privacy notice included
- [ ] Consent checkbox (if collecting PII)
- [ ] GDPR compliance (right to deletion noted)
- [ ] Data retention policy stated
- [ ] No collection of sensitive data without justification

## Outreach
- [ ] Subject line is compelling (not spammy)
- [ ] Clear value proposition for respondent
- [ ] Estimated time to complete stated
- [ ] Incentive disclosed (if any)
- [ ] Unsubscribe option (email)
- [ ] Test send completed (no broken links)

## Response Quality
- [ ] Screener questions filter right audience
- [ ] Attention check question included (optional)
- [ ] Response rate tracked
- [ ] Sample size achieved or timeline adjusted
- [ ] Data export verified (CSV/JSON valid)

## Analysis
- [ ] Data cleaned (remove test responses, incomplete)
- [ ] Segment analysis performed
- [ ] Statistical significance noted
- [ ] Qualitative themes identified
- [ ] Insights actionable (not just descriptive)

## Documentation
- [ ] survey_design.md complete
- [ ] analysis_report.md written
- [ ] action_items.md prioritized
- [ ] Raw data stored securely
