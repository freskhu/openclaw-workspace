# Survey Outreach - Prompt Template

```
Execute survey outreach with the following parameters:

SURVEY_GOAL: {{survey_goal}}
TARGET_AUDIENCE: {{target_audience}}
CHANNEL: {{channel}}
SAMPLE_SIZE: {{sample_size}}
TIMELINE: {{timeline}}

TASKS:
1. Design survey questions:
   - Screeners (if needed)
   - Core questions (max 10 for attention)
   - Demographics
   - Open-ended feedback
2. Review for bias (no leading questions)
3. Set up survey in tool (Typeform/Google Forms)
4. Prepare outreach message
5. Execute outreach (send emails/announce)
6. Monitor response rate
7. Analyze results when sample achieved
8. Generate insights and action items

CONSTRAINTS:
- Max 10 questions (keep it short)
- No required fields except consent
- Mobile-friendly design
- GDPR/privacy compliant
- Clear value proposition for respondent

DELIVERABLES:
- Survey design document
- Response data (anonymized)
- Analysis report with insights
- Action items prioritized
```
